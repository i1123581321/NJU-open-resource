# Basic Operators

`=` 没有返回值，以避免和 `==` 混用

算数操作符会检测并避免 overflow

swift 提供了范围操作符 `a..<b`（左闭右开区间）和 `a...b` （闭区间）

## terminology

* 一元操作符，分为前缀和后缀
* 二元操作符，中缀
* 三元操作符，只有一个 `a ? b : c`

## 赋值操作符

`=` 可以初始化或更新一个变量的值

如果右边是一个 tuple 的话，可以分解到多个变量或常量

```swift
let (x, y) = (1, 2)
// x is equal to 1, and y is equal to 2
```

`=` 没有返回值，用作 if 的条件时会产生错误

## 算术操作符

`+ - * /`

不允许溢出，如果需要溢出功能可以使用溢出操作符，如 `a &+ b`

`+` 同样支持字符串的连接

`%` 取余

一元的前缀操作符 `-` 可以取负，而 `+` 不改变变量的值
